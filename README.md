# 📊 FITNESS & HEALTH ANALYTICS DASHBOARD

An interactive **Power BI Dashboard** designed to analyze **fitness behavior, health patterns, and lifestyle metrics**.

This project provides meaningful insights into **user activity, calories burned, sleep habits, and workout performance** to support **data-driven decision making**.

---

# 🚀 PROJECT OVERVIEW

The goal of this project is to understand how **daily habits impact health and fitness outcomes**.

The dashboard focuses on:

• 📈 Activity & performance trends  
• 🔥 Calorie burn analysis  
• 😴 Sleep vs activity relationship  
• 🏋️ Workout behavior insights  
• 🥗 Diet & lifestyle patterns  

---

# 📌 KEY KPIs

• 👥 **Total Users**  
• 🔥 **Total Calories Burned**  
• 🚶 **Total Daily Steps**  
• 😴 **Average Sleep Hours**  
• ⏱ **Average Workout Duration**

---

# 📊 DASHBOARD VISUALS

The dashboard includes:

📉 **Calories Burned by Activity Type**  
📈 **Daily Steps vs Calories Burned (Scatter Plot)**  
📊 **Workout Duration by Age Group**  
🥗 **Calories by Diet Preference (Donut Chart)**  
📈 **Sleep Hours Trend Analysis**  
🎯 **Slicers for Gender, Age, Activity & Diet**

---

# 📂 DATASET FEATURES

The dataset contains:

• User_ID  
• Age Group  
• Gender  
• Activity Type  
• Daily Steps  
• Calories Burned  
• Hours of Sleep  
• Workout Duration
